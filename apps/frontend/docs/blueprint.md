# **App Name**: EvoAvatar

## Core Features:

- Avatar Invitation: Allow the user to invite one trusted participant to train their AI avatar.
- Continuous Learning: Enable the AI avatar to continuously learn from text and voice inputs.
- Structured Memory: Implement structured memories to reflect evolving opinions and behaviors, for use by a RAG tool.
- Avatar Training Interface: Provide an interface for training the avatar via text and voice inputs.
- Chat Interface: Enable users to chat with their AI avatar and provide feedback.
- 3D Avatar Rotation: Display the avatar with 360-degree rotation on the landing page.
- Profile Page: User profile page to visualize the avator, start new training and review memories.

## Style Guidelines:

- Primary color: Saturated purple (#A64AC9) to convey cutting-edge technology.
- Background color: Dark grey (#1A1A1A) for a modern and classy aesthetic.
- Accent color: Cyan (#4AC9C6) for highlights and interactive elements. It is close to primary color but clearly differentiated in brightness and saturation.
- Implement a color controller with options: red (#FF0000), orange (#FFA500), sky blue (#87CEEB), cyan (#00FFFF), pink (#FF69B4), green (#00FF00); which change text colors across the website.
- Font: 'Tomorrow' (sans-serif) for all text elements to provide a modern feel. Note: currently only Google Fonts are supported.
- Design a clean and modern layout with a focus on user experience, particularly in the avatar training and chat interfaces.
- Incorporate subtle scrolling effects and parallax for an attractive landing page experience.
